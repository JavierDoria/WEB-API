# API RESTful con ASP.NET Core Web API. -CRUD

## Descripción
Desarrollo de una API RESTful utilizando ASP.NET Core Web API sobre .NET 8.0, implementando operaciones CRUD mediante métodos HTTP GET, POST, PUT y DELETE.
Es un sistema de boleteria basica, la cual se encargara del registro  de una persona para cierto evento, previamente el usuario tendra que ser registrado, para asi poder adquirir una entrada 

## Tecnologías utilizadas
- ASP.NET Core Web API / .NET 8
- Swagger
- C#
- SQL Server
- HTTP REST
